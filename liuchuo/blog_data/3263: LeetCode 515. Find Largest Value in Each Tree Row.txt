<p>You need to find the largest value in each row of a binary tree.</p><p>Example:<br>Input:</p><p>1<br>/ \\<br>3 2<br>/ \\ \\<br>5 3 9</p><p>Output: [1, 3, 9]</p><p><span style="color: #99cc00;"><strong>题目大意：找二叉树每一层最大的数～然后返回一个数组表示每一层的最大的数～</strong></span></p><p><span style="color: #ff9900;"><strong>分析：先用广度优先搜索对二叉树进行层序遍历，每一层设立maxn保存每一层的最大值，然后在每一层遍历完毕之后将maxn的值放入result数组中～</strong></span></p><p> </p>
