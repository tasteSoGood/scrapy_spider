<p>SQL语言用ALTER TABLE语句修改基本表，其一般格式为：</p><p>ALTER TABLE &lt;表名&gt;</p><p>[ADD [COLUMN] &lt;新列名&gt;&lt;数据类型&gt;[完整性约束]]</p><p>[ADD &lt;表级完整性约束&gt;]</p><p>[DROP [COLUMN] &lt;列名&gt; [CASCADE | RESTRICT]]</p><p>[DROP CONSTRAINT&lt;完整性约束名&gt; [RESTRICT | CASCADE]]</p><p>[ALTER COLUMN &lt;列名&gt;&lt;数据类型&gt;]</p><p>下面哪个SQL命令用来向表中添加列（ ）<br>A. MODIFY TABLE TableName ADD COLUMN ColumnName<br>B. MODIFY TABLE TableName ADD ColumnName<br>C. ALTER TABLE TableName ADD COLUMN ColumnName<br>D. ALTER TABLE TableName ADD ColumnName Type</p><p>【答案】：D</p><p>【解析】：由上面的语法格式可知排除A和B，修改列名的时候必须要指定数据类型，而C项没有指定列的数据类型，故这里选择D项</p>
