<p>Given two integers L and R, find the count of numbers in the range [L, R] (inclusive) having a prime number of set bits in their binary representation.<br>(Recall that the number of set bits an integer has is the number of 1s present when written in binary. For example, 21 written in binary is 10101 which has 3 set bits. Also, 1 is not a prime.)<br>Example 1:<br>Input: L = 6, R = 10<br>Output: 4<br>Explanation:<br>6 -&gt; 110 (2 set bits, 2 is prime)<br>7 -&gt; 111 (3 set bits, 3 is prime)<br>9 -&gt; 1001 (2 set bits , 2 is prime)<br>10-&gt;1010 (2 set bits , 2 is prime)<br>Example 2:<br>Input: L = 10, R = 15<br>Output: 5<br>Explanation:<br>10 -&gt; 1010 (2 set bits, 2 is prime)<br>11 -&gt; 1011 (3 set bits, 3 is prime)<br>12 -&gt; 1100 (2 set bits, 2 is prime)<br>13 -&gt; 1101 (3 set bits, 3 is prime)<br>14 -&gt; 1110 (3 set bits, 3 is prime)<br>15 -&gt; 1111 (4 set bits, 4 is not prime)<br>Note:<br>L, R will be integers L &lt;= R in the range [1, 10^6].<br>R – L will be at most 10000.</p><p><span style="color: #99cc00;"><strong>题目大意：给两个数L和R，在[L, R]区间寻找数字的二进制中1的个数是素数的数字个数。</strong></span></p><p><span style="color: #ff9900;"><strong>分析：R不超过10的6次方，也就是不超过2的20次方，那么判断是否为素数只需判断是否等于2 3 5 7 11 13 17 19即可，用bitset也只需20位表示即可，用count函数计算1的个数，然后用cnt统计后返回～</strong></span></p><p> </p>
