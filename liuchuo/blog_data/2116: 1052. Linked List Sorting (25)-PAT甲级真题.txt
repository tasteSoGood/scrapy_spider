NULL is represented by -1.</strong></span></p><p><span style="color: #33cccc;"><strong>Then N lines follow, each describes a node in the format:</strong></span></p><p><span style="color: #33cccc;"><strong>Address Key Next</strong></span></p><p><span style="color: #33cccc;"><strong>where Address is the address of the node in memory, Key is an integer in [-105, 105], and Next is the address of the next node. It is guaranteed that all the keys are distinct and there is no cycle in the linked list starting from the head node.</strong></span></p><p><span style="color: #3366ff;"><strong>Output Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>For each test case, the output format is the same as that of the input, where N is the total number of nodes in the list and all the nodes must be sorted order.</strong></span></p><p><span style="color: #3366ff;"><strong>Sample Input:</strong></span><br><span style="color: #33cccc;"><strong>5 00001</strong></span><br><span style="color: #33cccc;"><strong>11111 100 -1</strong></span><br><span style="color: #33cccc;"><strong>00001 0 22222</strong></span><br><span style="color: #33cccc;"><strong>33333 100000 11111</strong></span><br><span style="color: #33cccc;"><strong>12345 -1 33333</strong></span><br><span style="color: #33cccc;"><strong>22222 1000 12345</strong></span><br><span style="color: #3366ff;"><strong>Sample Output:</strong></span><br><span style="color: #33cccc;"><strong>5 12345</strong></span><br><span style="color: #33cccc;"><strong>12345 -1 00001</strong></span><br><span style="color: #33cccc;"><strong>00001 0 11111</strong></span><br><span style="color: #33cccc;"><strong>11111 100 22222</strong></span><br><span style="color: #33cccc;"><strong>22222 1000 33333</strong></span><br><span style="color: #33cccc;"><strong>33333 100000 -1</strong></span></p><p><span style="color: #99cc00;"><strong>题目大意：给出一个链表，将链表排序，然后把链表上的结点按照data值的从小到大顺序输出</strong></span><br><span style="color: #ff9900;"><strong>分析：建立结构体数组，按照从首地址开始的顺序（直到-1）遍历一遍整个链表，将在链表中的结点的flag标记为true，并且统计cnt（有效结点的个数）。（因为有的结点根本不在链表中）</strong></span><br><span style="color: #ff9900;"><strong>然后将链表进行排序，如果flag == false就把他们移动到后面（即：reuturn a.flag &gt; b.flag）,最后只输出前cnt个链表的信息～</strong></span></p><p> </p>
