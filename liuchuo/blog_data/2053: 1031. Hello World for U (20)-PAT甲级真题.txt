<p><span style="color: #33cccc;"><strong>Given any string of N (&gt;=5) characters, you are asked to form the characters into the shape of U. For example, “helloworld” can be printed as:</strong></span><br><span style="color: #33cccc;"><strong>h    d</strong></span><br><span style="color: #33cccc;"><strong>e     l</strong></span><br><span style="color: #33cccc;"><strong>l     r</strong></span><br><span style="color: #33cccc;"><strong>lowo</strong></span><br><span style="color: #33cccc;"><strong>That is, the characters must be printed in the original order, starting top-down from the left vertical line with n1 characters, then left to right along the bottom line with n2 characters, and finally bottom-up along the vertical line with n3 characters. And more, we would like U to be as squared as possible — that is, it must be satisfied that n1 = n3 = max { k| k &lt;= n2 for all 3 &lt;= n2 &lt;= N } with n1 + n2 + n3 – 2 = N.</strong></span></p><p><span style="color: #3366ff;"><strong>Input Specification:</strong></span><br><span style="color: #33cccc;"><strong>Each input file contains one test case. Each case contains one string with no less than 5 and no more than 80 characters in a line. The string contains no white space.</strong></span></p><p><span style="color: #3366ff;"><strong>Output Specification:</strong></span><br><span style="color: #33cccc;"><strong>For each test case, print the input string in the shape of U as specified in the description.</strong></span></p><p><span style="color: #3366ff;"><strong>Sample Input:</strong></span><br><span style="color: #33cccc;"><strong>helloworld!</strong></span></p><p><span style="color: #3366ff;"><strong>Sample Output:</strong></span><br><span style="color: #33cccc;"><strong>h      !</strong></span><br><span style="color: #33cccc;"><strong>e      d</strong></span><br><span style="color: #33cccc;"><strong>l       l</strong></span><br><span style="color: #33cccc;"><strong>lowor</strong></span></p><p><span style="color: #99cc00;"><strong>题目大意：用所给字符串按U型输出。n1和n3是左右两条竖线从上到下的字符个数，n2是底部横线从左到右的字符个数。</strong></span><br><span style="color: #99cc00;"><strong>要求：</strong></span><br><span style="color: #99cc00;"><strong>1. n1 == n3</strong></span><br><span style="color: #99cc00;"><strong>2. n2 &gt;= n1</strong></span><br><span style="color: #99cc00;"><strong>3. n1为在满足上述条件的情况下的最大值</strong></span></p><p><span style="color: #ff9900;"><strong>分析：假设n = 字符串长度 + 2，因为2 * n1 + n2 = n，且要保证n2 &gt;= n1, n1尽可能地大，分类讨论：</strong></span><br><span style="color: #ff9900;"><strong>1. 如果n % 3 == 0，n正好被3整除，直接n1 == n2 == n3;</strong></span><br><span style="color: #ff9900;"><strong>2. 如果n % 3 == 1，因为n2要比n1大，所以把多出来的那1个给n2</strong></span><br><span style="color: #ff9900;"><strong>3. 如果n % 3 == 2, 就把多出来的那2个给n2</strong></span><br><span style="color: #ff9900;"><strong>所以得到公式：n1 = n / 3，n2 = n / 3 + n % 3</strong></span><br><span style="color: #ff9900;"><strong>把它们存储到二维字符数组中，一开始初始化字符数组为空格，然后按照u型填充进去，最后输出这个数组u～～</strong></span></p><p></p>
