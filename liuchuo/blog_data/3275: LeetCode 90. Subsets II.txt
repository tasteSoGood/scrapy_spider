<p>Given a collection of integers that might contain duplicates, nums, return all possible subsets.</p><p>Note: The solution set must not contain duplicate subsets.</p><p>For example,<br>If nums = [1,2,2], a solution is:</p><p>[<br>[2],<br>[1],<br>[1,2,2],<br>[2,2],<br>[1,2],<br>[]<br>]</p><p><span style="color: #99cc00;"><strong>题目大意：给一个集合nums，nums中可能含有重复元素，求nums的所有子集集合～集合中的元素不能重复～</strong></span></p><p><span style="color: #ff9900;"><strong>分析：首先对nums进行排序，接着用位运算，j从0到maxn变化，每一次计算j移动i位后最后一位是否为1，如果为1就将nums[i]的值放入result[j]～为了保证不重复，将result的结果放入集合s中，然后将s中的结果再放入result数组中返回～</strong></span></p><p> </p>
