<p>Given a m x n grid filled with non-negative numbers, find a path from top left to bottom right which minimizes the sum of all numbers along its path.</p><p>Note: You can only move either down or right at any point in time.</p><p><strong><span style="color: #ff9900;">分析：用一个dp二维数组标记当前方格的最小值～</span></strong></p><p><strong><span style="color: #ff9900;">0.当i == 0 &amp;&amp; j == 0的时候，dp[i][j] = grid[i][j];</span></strong></p><p><strong><span style="color: #ff9900;">1.对于i==0的时候，为最上面一排，当前方格只能由左边方格来，所以dp[i][j] = dp[i][j-1] + grid[i][j];</span></strong></p><p><strong><span style="color: #ff9900;">2.对于j==0的时候，为最左边一排，当前方格只能由上边方格来，所以dp[i][j] = dp[i-1][j] + grid[i][j];</span></strong></p><p><strong><span style="color: #ff9900;">3.其他情况下，dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j];</span></strong></p><p><strong><span style="color: #ff9900;">最后直到一直递推输出到终点(m-1, n-1)的时候return dp[m-1][n-1];～～</span></strong></p><p></p>
