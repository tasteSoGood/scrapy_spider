<p><span style="color: #33cccc;"><strong>Given a constant K and a singly linked list L, you are supposed to reverse the links of every K elements on L. For example, given L being 1→2→3→4→5→6, if K = 3, then you must output 3→2→1→6→5→4; if K = 4, you must output 4→3→2→1→5→6.</strong></span></p><p><span style="color: #3366ff;"><strong>Input Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>Each input file contains one test case. For each case, the first line contains the address of the first node, a positive N (&lt;= 105) which is the total number of nodes, and a positive K (&lt;=N) which is the length of the sublist to be reversed. The address of a node is a 5-digit nonnegative integer, and NULL is represented by -1.</strong></span><br><span style="color: #33cccc;"><strong>Then N lines follow, each describes a node in the format:</strong></span><br><span style="color: #33cccc;"><strong>Address Data Next</strong></span><br><span style="color: #33cccc;"><strong>where Address is the position of the node, Data is an integer, and Next is the position of the next node.</strong></span></p><p><span style="color: #3366ff;"><strong>Output Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>For each case, output the resulting ordered linked list. Each node occupies a line, and is printed in the same format as in the input.</strong></span></p><p><strong><span style="color: #3366ff;">Sample Input:</span></strong></p><p><span style="color: #33cccc;"><strong>00100 6 4<br>00000 4 99999<br>00100 1 12309<br>68237 6 -1<br>33218 3 00000<br>99999 5 68237<br>12309 2 33218</strong></span></p><p><strong><span style="color: #3366ff;">Sample Output:</span></strong></p><p><span style="color: #33cccc;"><strong>00000 4 33218<br>33218 3 12309<br>12309 2 00100<br>00100 1 99999<br>99999 5 68237<br>68237 6 -1</strong></span></p><p><span style="color: #ff9900;"><strong>分析：把地址为temp的数的数值存入data[temp]中，把temp的下一个结点的地址存入next[temp]中。</strong></span><br><span style="color: #ff99cc;"><strong>注意：不一定所有的输入的结点都是有用的，加个计数器sum</strong></span></p><p> </p>
