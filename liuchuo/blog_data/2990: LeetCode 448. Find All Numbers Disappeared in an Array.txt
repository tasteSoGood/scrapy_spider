<p>Given an array of integers where 1 ≤ a[i] ≤ n (n = size of array), some elements appear twice and others appear once.</p><p>Find all the elements of [1, n] inclusive that do not appear in this array.</p><p>Could you do it without extra space and in O(n) runtime? You may assume the returned list does not count as extra space.</p><p>Example:</p><p>Input:<br>[4,3,2,7,8,2,3,1]</p><p>Output:<br>[5,6]</p><p><span style="color: #ff9900;"><strong>分析：hash[i]用来标记i是否在nums数组中出现过。然后将没有出现过的放入result数组中，返回result数组～～～</strong></span></p><p> </p>
