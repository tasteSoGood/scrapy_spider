<p><span style="color: #3366ff;"><strong>1010. Radix (25)</strong></span><br><span style="color: #33cccc;"><strong>Given a pair of positive integers, for example, 6 and 110, can this equation 6 = 110 be true? The answer is “yes”, if 6 is a decimal number and 110 is a binary number.</strong></span></p><p><span style="color: #33cccc;"><strong>Now for any pair of positive integers N1 and N2, your task is to find the radix of one number while that of the other is given.</strong></span></p><p><span style="color: #3366ff;"><strong>Input Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>Each input file contains one test case. Each case occupies a line which contains 4 positive integers:</strong></span><br><span style="color: #33cccc;"><strong>N1 N2 tag radix</strong></span><br><span style="color: #33cccc;"><strong>Here N1 and N2 each has no more than 10 digits. A digit is less than its radix and is chosen from the set {0-9, a-z} where 0-9 represent the decimal numbers 0-9, and a-z represent the decimal numbers 10-35. The last number “radix” is the radix of N1 if “tag” is 1, or of N2 if “tag” is 2.</strong></span></p><p><span style="color: #3366ff;"><strong>Output Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>For each test case, print in one line the radix of the other number so that the equation N1 = N2 is true.
