<p>Given a positive integer n, find the least number of perfect square numbers (for example, 1, 4, 9, 16, …) which sum to n.<br>For example, given n = 12, return 3 because 12 = 4 + 4 + 4; given n = 13, return 2 because 13 = 4 + 9.</p><p><strong><span style="color: #ff9900;">分析：建立一个n+1长度的数组dp，dp[i]表示i这个数构成平方和需要数字的最小个数。</span></strong><br><strong><span style="color: #ff9900;">当j*j&lt;i的时候，temp中保存j从1开始每加1得到的dp[i-j*j] + 1的最小值～</span></strong><br><strong><span style="color: #ff9900;">当j*j=i的时候，dp[i]的值就直接为1～</span></strong><br><strong><span style="color: #ff9900;">从2一直到n可以计算出dp[i]的所有值～</span></strong><br><strong><span style="color: #ff9900;">最后return dp[n]的值～</span></strong></p><p> </p>
