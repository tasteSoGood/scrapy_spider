<p>The Hamming distance between two integers is the number of positions at which the corresponding bits are different.</p><p>Given two integers x and y, calculate the Hamming distance.</p><p>Note:<br>0 ≤ x, y &lt; 2^31.</p><p>Example:</p><p>Input: x = 1, y = 4</p><p>Output: 2</p><p>Explanation:<br>1 (0 0 0 1)<br>4 (0 1 0 0)<br>↑ ↑</p><p>The above arrows point to positions where the corresponding bits are different.</p><p><span style="color: #ff9900;"><strong>分析：将x和y不断右移一位，然后比较他们的最后一位。对于右移一位，采用x / 2和 y / 2的方式，对于比较最后一位，即比较x % 2 和 y % 2，统计不相同的次数cnt，直到x和y都等于0为止</strong></span></p><p> </p>
