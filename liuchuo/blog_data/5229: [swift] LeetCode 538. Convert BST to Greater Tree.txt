<p>Given a Binary Search Tree (BST), convert it to a Greater Tree such that every key of the original BST is changed to the original key plus sum of all keys greater than the original key in BST.</p><p>Example:</p><p>Input: The root of a Binary Search Tree like this:<br>5<br>/ \\<br>2 13</p><p>Output: The root of a Greater Tree like this:<br>18<br>/ \\<br>20 13</p><p></p>
