<p><span style="color: #3366ff;"><strong>1007. Maximum Subsequence Sum (25)</strong></span><br><span style="color: #33cccc;"><strong>Given a sequence of K integers { N1, N2, …, NK }. A continuous subsequence is defined to be { Ni, Ni+1, …, Nj } where 1 &lt;= i &lt;= j &lt;= K. The Maximum Subsequence is the continuous subsequence which has the largest sum of its elements. For example, given sequence { -2, 11, -4, 13, -5, -2 }, its maximum subsequence is { 11, -4, 13 } with the largest sum being 20.</strong></span></p><p><span style="color: #33cccc;"><strong>Now you are supposed to find the largest sum, together with the first and the last numbers of the maximum subsequence.</strong></span></p><p><span style="color: #3366ff;"><strong>Input Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>Each input file contains one test case. Each case occupies two lines. The first line contains a positive integer K (&lt;= 10000). The second line contains K numbers, separated by a space.</strong></span></p><p><span style="color: #3366ff;"><strong>Output Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>For each test case, output in one line the largest sum, together with the first and the last numbers of the maximum subsequence. The numbers must be separated by one space, but there must be no extra space at the end of a line. In case that the maximum subsequence is not unique, output the one with the smallest indices i and j (as shown by the sample case). If all the K numbers are negative, then its maximum sum is defined to be 0, and you are supposed to output the first and the last numbers of the whole sequence.</strong></span></p><p><span style="color: #3366ff;"><strong>Sample Input:</strong></span><br><span style="color: #33cccc;"><strong>10</strong></span><br><span style="color: #33cccc;"><strong>-10 1 2 3 4 -5 -23 3 7 -21</strong></span><br><span style="color: #3366ff;"><strong>Sample Output:</strong></span><br><span style="color: #33cccc;"><strong>10 1 4</strong></span><br><span style="color: #99cc00;"><strong>题目大意：求最大连续子序列和，输出最大的和以及这个子序列的开始值和结束值。如果所有数都小于0，那么认为最大的和为0，并且输出首尾元素。</strong></span></p><p><span style="color: #ff99cc;"><strong>PS：感谢github用户<a style="color: #ff99cc;" href="https://github.com/fs19910227">@fs19910227</a>提供的pull request～</strong></span></p><p> </p>
