<p>A sequence of number is called arithmetic if it consists of at least three elements and if the difference between any two consecutive elements is the same.</p><p>For example, these are arithmetic sequence:</p><p>1, 3, 5, 7, 9<br>7, 7, 7, 7<br>3, -1, -5, -9<br>The following sequence is not arithmetic.</p><p>1, 1, 2, 5, 7</p><p>A zero-indexed array A consisting of N numbers is given. A slice of that array is any pair of integers (P, Q) such that 0 &lt;= P &lt; Q &lt; N.</p><p>A slice (P, Q) of array A is called arithmetic if the sequence:<br>A[P], A[p + 1], …, A[Q – 1], A[Q] is arithmetic. In particular, this means that P + 1 &lt; Q.</p><p>The function should return the number of arithmetic slices in the array A.<br>Example:</p><p>A = [1, 2, 3, 4]</p><p>return: 3, for 3 arithmetic slices in A: [1, 2, 3], [2, 3, 4] and [1, 2, 3, 4] itself.</p><p><span style="color: #99cc00;"><strong>题目大意：求一个数组中能构成至少连续三个数的等差数列的个数</strong></span></p><p><span style="color: #ff9900;"><strong>分析：有一个规律是：</strong></span><br><span style="color: #ff9900;"><strong>连续3个数构成等差数列，能组成的等差数列个数是1</strong></span><br><span style="color: #ff9900;"><strong>连续4个数构成等差数列，能组成的等差数列个数是3（1+2）</strong></span><br><span style="color: #ff9900;"><strong>连续5个数构成等差数列，能组成的等差数列个数是6（1+2+3）</strong></span><br><span style="color: #ff9900;"><strong>……</strong></span><br><span style="color: #ff9900;"><strong>所以每次当有三个构成等差数列的时候，cnt = 1，之后每连续增加一个数满足与之前的数构成等差数列，就令cnt++，并且将结果加到最终的result里面～</strong></span></p><p> </p>
