<p>Given an array of integers with possible duplicates, randomly output the index of a given target number. You can assume that the given target number must exist in the array.</p><p>Note:<br>The array size can be very large. Solution that uses too much extra space will not pass the judge.</p><p>Example:</p><p>int[] nums = new int[] {1,2,3,3,3};<br>Solution solution = new Solution(nums);</p><p>// pick(3) should return either index 2, 3, or 4 randomly. Each index should have equal probability of returning.<br>solution.pick(3);</p><p>// pick(1) should return 0. Since in the array only nums[0] is equal to 1.<br>solution.pick(1);</p><p><span style="color: #99cc00;"><strong>题目大意：给一个整数数组，里面可能有重复元素。给一个target，随机返回一个数组元素等于target的元素下标～</strong></span></p><p><span style="color: #ff9900;"><strong>分析：将所有等于target的元素的下标存储在index数组中，然后根据rand() % index.size()的值随机从index中取一个值返回～</strong></span></p><p> </p>
