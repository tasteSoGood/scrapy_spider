<p><span style="color: #33cccc;"><strong>Behind the scenes in the computer’s memory, color is always talked about as a series of 24 bits of information for each pixel. In an image, the color with the largest proportional area is called the dominant color. A strictly dominant color takes more than half of the total area. Now given an image of resolution M by N (for example, 800×600), you are supposed to point out the strictly dominant color.</strong></span></p><p><span style="color: #3366ff;"><strong>Input Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>Each input file contains one test case. For each case, the first line contains 2 positive numbers: M (&lt;=800) and N (&lt;=600) which are the resolutions of the image. Then N lines follow, each contains M digital colors in the range [0, 224). It is guaranteed that the strictly dominant color exists for each input image. All the numbers in a line are separated by a space.</strong></span></p><p><span style="color: #3366ff;"><strong>Output Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>For each test case, simply print the dominant color in a line.</strong></span></p><p><span style="color: #3366ff;"><strong>Sample Input:</strong></span><br><span style="color: #33cccc;"><strong>5 3</strong></span><br><span style="color: #33cccc;"><strong>0 0 255 16777215 24</strong></span><br><span style="color: #33cccc;"><strong>24 24 0 0 24</strong></span><br><span style="color: #33cccc;"><strong>24 0 24 24 24</strong></span><br><span style="color: #3366ff;"><strong>Sample Output:</strong></span><br><span style="color: #33cccc;"><strong>24</strong></span></p><p><span style="color: #ff9900;"><strong>题目大意：选取主色调，就是M列N行的矩阵里面出现次数多余一半的那个数字～</strong></span></p><p><span style="color: #ff9900;"><strong>分析：STL中map的应用～</strong></span><span style="color: #ff9900;"><strong>使用arr[i] = j表示i元素在矩阵中出现了j次，在输入的同时比较arr当前的值是否已经超过半数，如果超过，就直接输出该数字并退出程序～<br></strong></span></p><p> </p>
