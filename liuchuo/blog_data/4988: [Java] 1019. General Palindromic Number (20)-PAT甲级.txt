<p><span style="color: #3366ff;"><strong>1019. General Palindromic Number (20)</strong></span><br><span style="color: #33cccc;"><strong>A number that will be the same when it is written forwards or backwards is known as a Palindromic Number. For example, 1234321 is a palindromic number. All single digit numbers are palindromic numbers.</strong></span></p><p><span style="color: #33cccc;"><strong>Although palindromic numbers are most often considered in the decimal system, the concept of palindromicity can be applied to the natural numbers in any numeral system. Consider a number N &gt; 0 in base b &gt;= 2, where it is written in standard notation with k+1 digits ai as the sum of (aibi) for i from 0 to k. Here, as usual, 0 &lt;= ai &lt; b for all i and ak is non-zero. Then N is palindromic if and only if ai = ak-i for all i. Zero is written 0 in any base and is also palindromic by definition.</strong></span></p><p><span style="color: #33cccc;"><strong>Given any non-negative decimal integer N and a base b, you are supposed to tell if N is a palindromic number in base b.</strong></span></p><p><span style="color: #3366ff;"><strong>Input Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>Each input file contains one test case. Each case consists of two non-negative numbers N and b, where 0 &lt;= N &lt;= 109 is the decimal number and 2 &lt;= b &lt;= 109 is the base. The numbers are separated by a space.</strong></span></p><p><span style="color: #3366ff;"><strong>Output Specification:</strong></span></p><p><span style="color: #33cccc;"><strong>For each test case, first print in one line “Yes” if N is a palindromic number in base b, or “No” if not. Then in the next line, print N as the number in base b in the form “ak ak-1 … a0”. Notice that there must be no extra space at the end of output.</strong></span></p><p><span style="color: #3366ff;"><strong>Sample Input 1:</strong></span><br><span style="color: #33cccc;"><strong>27 2</strong></span><br><span style="color: #3366ff;"><strong>Sample Output 1:</strong></span><br><span style="color: #33cccc;"><strong>Yes</strong></span><br><span style="color: #33cccc;"><strong>1 1 0 1 1</strong></span><br><span style="color: #3366ff;"><strong>Sample Input 2:</strong></span><br><span style="color: #33cccc;"><strong>121 5</strong></span><br><span style="color: #3366ff;"><strong>Sample Output 2:</strong></span><br><span style="color: #33cccc;"><strong>No</strong></span><br><span style="color: #33cccc;"><strong>4 4 1</strong></span></p><p><span style="color: #ff9900;"><strong>题目大意：给出两个整数a和b，问十进制的a在b进制下是否为回文数。是的话输出Yes，不是输出No。并且输出a在b进制下的表示，以空格隔开</strong></span></p><p><span style="color: #ff99cc;"><strong>PS：感谢github用户<a style="color: #ff99cc;" href="https://github.com/fs19910227">@fs19910227</a>提供的pull request～</strong></span></p><p> </p>
