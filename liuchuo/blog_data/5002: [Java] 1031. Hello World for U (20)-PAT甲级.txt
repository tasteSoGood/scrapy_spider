<p><span style="color: #3366ff;"><strong>1031. Hello World for U (20)</strong></span><br><span style="color: #33cccc;"><strong>Given any string of N (&gt;=5) characters, you are asked to form the characters into the shape of U. For example, “helloworld” can be printed as:</strong></span><br><span style="color: #33cccc;"><strong>h    d</strong></span><br><span style="color: #33cccc;"><strong>e     l</strong></span><br><span style="color: #33cccc;"><strong>l     r</strong></span><br><span style="color: #33cccc;"><strong>lowo</strong></span><br><span style="color: #33cccc;"><strong>That is, the characters must be printed in the original order, starting top-down from the left vertical line with n1 characters, then left to right along the bottom line with n2 characters, and finally bottom-up along the vertical line with n3 characters. And more, we would like U to be as squared as possible — that is, it must be satisfied that n1 = n3 = max { k| k &lt;= n2 for all 3 &lt;= n2 &lt;= N } with n1 + n2 + n3 – 2 = N.</strong></span></p><p><span style="color: #3366ff;"><strong>Input Specification:</strong></span><br><span style="color: #33cccc;"><strong>Each input file contains one test case. Each case contains one string with no less than 5 and no more than 80 characters in a line. The string contains no white space.</strong></span></p><p><span style="color: #3366ff;"><strong>Output Specification:</strong></span><br><span style="color: #33cccc;"><strong>For each test case, print the input string in the shape of U as specified in the description.</strong></span></p><p><span style="color: #3366ff;"><strong>Sample Input:</strong></span><br><span style="color: #33cccc;"><strong>helloworld!</strong></span></p><p><span style="color: #3366ff;"><strong>Sample Output:</strong></span><br><span style="color: #33cccc;"><strong>h      !</strong></span><br><span style="color: #33cccc;"><strong>e      d</strong></span><br><span style="color: #33cccc;"><strong>l       l</strong></span><br><span style="color: #33cccc;"><strong>lowor</strong></span></p><p><span style="color: #99cc00;"><strong>题目大意：用所给字符串按U型输出。n1和n3是左右两条竖线从上到下的字符个数，n2是底部横线从左到右的字符个数。</strong></span><br><span style="color: #99cc00;"><strong>要求：</strong></span><br><span style="color: #99cc00;"><strong>1. n1 == n3</strong></span><br><span style="color: #99cc00;"><strong>2. n2 &gt;= n1</strong></span><br><span style="color: #99cc00;"><strong>3. n1为在满足上述条件的情况下的最大值</strong></span></p><p><span style="color: #ff99cc;"><strong>PS：感谢github用户<a style="color: #ff99cc;" href="https://github.com/fs19910227">@fs19910227</a>提供的pull request～</strong></span></p><p></p>
