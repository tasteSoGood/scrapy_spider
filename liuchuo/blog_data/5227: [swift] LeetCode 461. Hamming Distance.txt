<p><strong>The Hamming distance between two integers is the number of positions at which the corresponding bits are different.</strong></p><p><strong>Given two integers x and y, calculate the Hamming distance.</strong></p><p><strong>Note:</strong><br><strong>0 ≤ x, y &lt; 2^31.</strong></p><p><strong>Example:</strong></p><p><strong>Input: x = 1, y = 4</strong></p><p><strong>Output: 2</strong></p><p><strong>Explanation:</strong><br><strong>1 (0 0 0 1)</strong><br><strong>4 (0 1 0 0)</strong><br><strong>↑ ↑</strong></p><p><strong>The above arrows point to positions where the corresponding bits are different.</strong></p><p></p>
