<p>Given a non-empty special binary tree consisting of nodes with the non-negative value, where each node in this tree has exactly two or zero sub-node. If the node has two sub-nodes, then this node’s value is the smaller value among its two sub-nodes.</p><p>Given such a binary tree, you need to output the second minimum value in the set made of all the nodes’ value in the whole tree.</p><p>If no such second minimum value exists, output -1 instead.</p><p>Example 1:<br>Input:<br>2<br>/ \\<br>2 5<br>/ \\<br>5 7</p><p>Output: 5<br>Explanation: The smallest value is 2, the second smallest value is 5.<br>Example 2:<br>Input:<br>2<br>/ \\<br>2 2</p><p>Output: -1<br>Explanation: The smallest value is 2, but there is not any second smallest value.</p><p><span style="color: #99cc00;"><strong>题目大意：给一个非空的树，这个树所有值非负，而且每个结点只有2个孩子或者0个孩子，如果当前结点有两个孩子，则这个结点的值比它的两个孩子的值都小，要求输出这个树第二小的值。</strong></span></p><p><span style="color: #ff9900;"><strong>分析：遍历树，将所有的值放入集合中，输出集合的第二个数；如果集合的大小小于等于1，则输出-1</strong></span></p><p> </p>
